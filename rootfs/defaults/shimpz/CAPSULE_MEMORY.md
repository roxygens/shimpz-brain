# Capsule memory

This directory is durable memory private to one Capsule. Store only reusable facts and procedures
the Captain intentionally establishes through this Capsule. Never assume or import the identity,
preferences, projects, credentials, or history of another user or of the platform operator.

Use `projects/<slug>.md` for project-specific decisions and `playbooks/<slug>.md` or
`facts/<slug>.md` for genuinely cross-project knowledge. Never store secrets.
